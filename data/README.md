## 挂载目录

### mysql

    - conf/my.cnf           配置文件
    - data                  数据存放
    - log
        - error.log         错误日志
        - slow_query.log    慢查询日志
        
### redis
    
    - conf/redis.conf       配置文件
    - data                  数据存放
    
### php

    - logs                  错误日志
        - error.log
    - php.ini
    - php-fpm.conf
    - www.conf
    
### nginx

    - logs                  日志
        - access.log
        - error.log     
    - conf                  配置文件
        - nginx.conf
        - vhost
            - www.wangyongdong.com.conf
            
### www
    - 网站代码存放位置
